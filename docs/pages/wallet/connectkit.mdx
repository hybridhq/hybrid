# ConnectKit + Hybrid

How to use ConnectKit as your wallet connect library:

## 1. Install ConnectKit

```sh
npm add hybrid-ConnectKit
```

View the full instructions on [ConnectKit Docs](https://docs.family.co/connectkit)

## 2. Configure ConnectKit wallet and send to Web3Provider

Edit your `pages/_app.tsx`

```tsx
import { Web3Provider } from "hybrid"
import { ConnectKit } from "hybrid-ConnectKit"

const wallet = ConnectKit({
	appName: "Hybrid App"
})

const App = ({ Component, pageProps }) => {
	return (
		<Web3Provider wallet={wallet}>
			<Component {...pageProps} />
		</Web3Provider>
	)
}

export default App
```

## 3. Use ConnectKit Components in your dapp:

```tsx
import { ConnectButton } from "@rainbow-me/ConnectKit"

const MyComponent = () => {
	return (
		<div>
			<ConnectButton />
		</div>
	)
}

export default App
```
