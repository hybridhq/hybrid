# What is MintDrop?

MintDrop is your complete NFT development stack. Quickly and easily develop,
test, deploy, and integrate NFTs into any application.

- [Foundry](https://getfoundry.sh/) for smart contract development.
- [Wagmi](https://wagmi.sh/) and [Ethers](https://docs.ethers.org/v5/) for contract interaction.
- [Connectkit](https://docs.family.co/connectkit) or [Rainbowkit](https://www.rainbowkit.com/) for wallet connectivity.
  {/* - [Ponder](https://ponder.sh/) for smart contract indexing. */}

Additionally, MintDrop comes with React tooling and components to make working
with deployed smart contracts much easier.

## Quickstart

```sh
npx mintdrop@latest init
```

Then run the MintDrop dev server:

```sh
mint dev
```
