# Rainbowkit + Hybrid

How to use Rainbowkit as your wallet connect library:

## 1. Install Rainbowkit

```sh
npm add hybrid-rainbowkit
```

View the full instructions on [RainbowKit Docs](https://www.rainbowkit.com/docs/installation#manual-setup)

## 2. Configure Rainbowkit wallet and send to Web3Provider

Edit your `pages/_app.tsx`

```tsx
import { Web3Provider } from "hybrid"
import { RainbowKit } from "hybrid-rainbowkit"

const wallet = RainbowKit({
	appName: "RainbowKit App"
})

const App = ({ Component, pageProps }) => {
	return (
		<Web3Provider wallet={wallet}>
			<Component {...pageProps} />
		</Web3Provider>
	)
}

export default App
```

## 3. Use Rainbowkit Components in your dapp:

```tsx
import { ConnectButton } from "@rainbow-me/rainbowkit"

const MyComponent = () => {
	return (
		<div>
			<ConnectButton />
		</div>
	)
}

export default App
```
