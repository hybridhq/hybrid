# \<MintButton\>

A context-aware generic MintButton for NFT smart contracts.

## Usage

```tsx
import { MintButton } from "hybrid"
import { Deployments } from "~/.hybrid/client"

const MyComponent = () => {
	return (
		<>
			<MintButton contract={Deployments.NFT} />
		</>
	)
}

export default App
```
