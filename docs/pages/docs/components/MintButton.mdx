# \<MintButton\>

A context-aware generic MintButton for NFT smart contracts.

## Usage

```tsx
import { MintButton } from "mintdrop/react"

const MyComponent = () => {
	return (
		<>
			<MintButton />
		</>
	)
}

export default App
```
