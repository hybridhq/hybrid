# \<ConnectedAs\>

Display who the connected wallet is, and links to Etherscan.

## Usage

```tsx
import { ConnectedAs } from "hybrid"

const MyComponent = () => {
	return (
		<>
			<ConnectedAs />
		</>
	)
}

export default App
```
